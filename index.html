Simplify your UI with styleSUP!

User-friendly library to develop quicker and easier user interface. Spend less time in designing and concentrate more on logic without hustling on UI.

How to use?

There are 2 ways to include this libary into your webpage.

  1. Inlcude the stylesheets link on html: "https://stylesup.netlify.app/styles/components.css"
  2. Import the stylesheet on css file @import url(https://stylesup.netlify.app/styles/components.css);
